# Chrome Web Store Listing - AI Live Terminal Bridge

## Short Description (132 characters max)
Captures console logs & network activity from localhost for AI debugging. Requires native host. Privacy-focused, localhost only.

## Detailed Description

**AI Live Terminal Bridge - Browser Monitor** is a developer tool that captures browser console logs, network activity, and errors from your localhost development environment and streams them to AI assistants for intelligent debugging.

### Key Features

✅ **Comprehensive Monitoring**
- Captures all console output (log, info, warn, error, debug)
- Monitors network requests (fetch, XMLHttpRequest)
- Tracks JavaScript errors with full stack traces
- Records performance metrics and timing data

✅ **Privacy & Security First**
- **Localhost only** - Only monitors localhost:* and 127.0.0.1:*
- **No external connections** - All data stays on your machine
- **Automatic redaction** - Cookies, tokens, and API keys are automatically removed
- **No tracking** - Zero analytics, no data collection, no third-party services
- **Open source** - Fully transparent and auditable code

✅ **Seamless Integration**
- **No UI required** - Runs silently in the background
- **Native messaging** - Connects to local MCP server via Chrome's native messaging
- **Real-time streaming** - Logs appear instantly as they happen
- **Development tunnels** - Supports ngrok, localtunnel, and Cloudflare tunnels

✅ **Developer-Friendly**
- Works with any localhost development server
- Zero configuration after setup
- Automatic reconnection if native host restarts
- Compatible with React, Vue, Angular, Next.js, and all web frameworks

### How It Works

1. **Install the extension** - Load it from the Chrome Web Store
2. **Install native host** - Follow the setup instructions to install the companion native messaging host
3. **Start developing** - Visit any localhost page in Chrome
4. **Automatic capture** - Extension monitors console, network, and errors
5. **AI debugging** - AI assistant can read logs to help debug issues

### Use Cases

**Perfect for developers who:**
- Use AI assistants (Claude, ChatGPT) for debugging help
- Want comprehensive logging without cluttering their code
- Need to share browser context with AI tools
- Debug complex frontend applications
- Track down elusive bugs in network requests
- Monitor performance issues in real-time

### Privacy Guarantee

This extension is designed with privacy as the top priority:

- **No data leaves your machine** - Everything is processed locally
- **No cloud services** - No external servers, APIs, or connections
- **No analytics** - We don't know if you installed it, we don't track usage
- **Complete control** - You own all logs, stored in your home directory
- **Open source** - Review the code yourself at https://github.com/Ami3466/ai-live-log-bridge

### Requirements

- Chrome browser (version 88 or later)
- Node.js installed (for native messaging host)
- AI Live Terminal Bridge MCP server (companion software)
- macOS or Linux (Windows support coming soon)

### Installation

1. Install this extension from the Chrome Web Store
2. Download and install the native messaging host from: https://github.com/Ami3466/ai-live-log-bridge
3. Follow the setup guide in the extension's README
4. Start developing on localhost and let the AI help you debug!
